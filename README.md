# Full Stack Open – Coursework
This repository contains my solutions and projects for the Full Stack Open course by the University of Helsinki.

## Structure
Each part of the course is organized into its own folder:

- part0 – Fundamentals
- part1 – React basics
- part2 – Communicating with server
- part3 – Node.js backend
- part4 – Testing and user management
- part5 – Frontend testing
- part6 – Redux

## Notes
Each exercise set is contained in its own subfolder.  
This repo follows the structure recommended by the course.

Course website: https://fullstackopen.com
